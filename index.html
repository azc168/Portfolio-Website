Weather Trend Display

This was a collaborative project I did for HackNC. Program scans in historical data from a file and produces a line graph of average maximum temperature, average minimum temperature, and average precipitations for each year. The average slope is also calculated and used to interpret the trends of the graph. These interpretations give program users more insight into what is going on and what actions should be taken from this information.

Nonograms Puzzle Application

A complete, functioning GUI implementation of the single-player logic puzzle nonograms. Nonograms are picture logic puzzles in which cells in a grid must be colored or left blank according to numbers at the side of the grid to reveal a hidden picture. I used the MVC and Observable Design Patterns to complete this project.
